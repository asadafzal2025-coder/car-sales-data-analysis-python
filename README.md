# Car Sales Data Analysis using Python

## Project Overview

This project demonstrates data cleaning, exploratory data analysis (EDA), and visualization of a car sales dataset using Python.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Project Objectives

- Explore the dataset
- Clean missing values
- Analyze pricing trends
- Study mileage distribution
- Compare manufacturers
- Visualize insights using charts

## Visualizations

- Brand Distribution
- Average Price by Brand
- Price Histogram
- Mileage vs Price
- Body Type Distribution

## Key Findings

- Compared average selling prices among manufacturers.
- Analyzed mileage and pricing trends.
- Identified the most common body types.
- Explored inventory distribution across brands.
